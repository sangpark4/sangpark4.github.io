기어박스 프로젝트

A gearbox system was designed to maximize the torque and speed of a small car. The system was a shifting one-stage gear network that consisted of gears, pinions, input and output shafts, bearings and housing. After calculations regarding the number of teeth and diameter of the gears and pinions, the system was modelled using Siemens NX and prepared for 3D printing.

식수 pH 제어 시스템

An unknown "black box" system was analyzed and controlled for this project. By using principles of control theory, the system was designed to control the quality of drinking water. In order to achieve this, we analyzed the controllability and stability of the system. Some of the tools that were used for this project include PID-controllers, Bode and Nyquist plots, transfer functions, and the linear time-invariance property.

Optimization of Identifying Deforested Regions in the Amazon Rainforest

An autonomous system that uses Lloyd's deployment algorithm was designed to program drones to effectively measure the density of vegetation in an area of the Amazon Rainforest. The goal for this project was to optimize the identification process to facilitate reforestation in the largest forest in the world.

Removal of Plastics in Sewage Systems

A filtration system was designed to eliminate macro and microplastics in drainage and sewage system. The designed solution used a dissolved air flotation process to effectively eliminate plastic in wastewater before it enters the treatment plant.

Sexual Assault Centre Kingston Guidance 모바일 앱

A mobile application was made using JavaScript and Android Studio in order to provide guidance to victims and educate other young adults. The target audience for this project were those aged 16 to 24, which was requested by Sexual Assault Centre Kingston.

The app consisted of four main sections: Resources, Scenarios, Consent, and Direct Questions. The Resources section was designed to provide the users with several helplines and sources to those in need. The Scenarios section is a quiz section that places the user in an observer's point of view where they witness an event taking place. The user would then choose an action they might take in order to help the victim. The Direct Questions section leads the user through a specific set of events that might have occurred to them. As the user specifies the situation more, the app will guide them to a resource that is suited for their circumstances. Finally, the Consent section provides general information regarding consent and the laws surrounding it.
